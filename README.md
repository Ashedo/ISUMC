# IMEConf 2025 Website

## Getting started

Ensure you have Node.js and npm installed.

Steps:

```sh
# 1) Clone the repository
git clone <YOUR_GIT_URL>

# 2) Install dependencies
npm i

# 3) Start the development server
npm run dev
```

## Tech stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
